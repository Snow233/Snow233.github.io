php序列化

因为 PHP 文件在执行结束以后就会将对象销毁，为了长久保存对象而提出的，下次使用只需要反序列化即可

目的是方便数据的传输和存储. json 是为了传递数据的方便性.

找到一篇非常好的文章PHP反序列化从初级到高级利用篇 - fish_pompom - 博客园
