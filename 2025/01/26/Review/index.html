Reverse部分

ida(最好用的反编译软件)

ida下载链接(https://hex-rays.com/) 官网免费下载

快捷键：

-F5/TAB 生成C语言伪代码
-U  undefine
-p  重新识别生成函数
-Shift+E  获取数据生成数组
-Shift+F12  汇总字符串String
-Alt+T  查找(等价于010Editor的Ctrl+F)
-F2  加断点

动态调试

linux端：
 将ida文件下IDA_Pro_7.7\dbgsrv文件夹下的linux_server（64）放到linux里
 查看linux的ip：ifconfig
 给权限：chmod 777 linux_server64
 运行：./linux_server64
windows端：
 f9：remote linux debugger
 hostname里填写刚刚查到的ip地址，其余不用管
 动调起来后就可以随意修改代码逻辑

Python反编译

提取pyc

python pyinstxtractor.py [文件完整路径]

pyc反编译

pycdc(.exe) [文件名] #括号内的部分可省略

pycdas查看字节码

pycdas(.exe) [文件名] #括号内的部分可省略
